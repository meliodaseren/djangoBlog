# Django Blog Demo

Python 3.6.3
Django 1.11.7

```shell
$ . venv/bin/activate
```

```shell
pip install django==1.11.7
pip freeze | grep Django
```

Run Server

```shell
$ python manage.py runserver 8000
```

後台管理頁面：`127.0.0.1:8000/admin`

Blog：`127.0.0.1:8000/blog`
